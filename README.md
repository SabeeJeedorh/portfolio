
Sabeea's Portfolio

A sleek and modern portfolio built with React and Tailwind CSS, showcasing my projects and skills.

Features

- Dark Mode/Light Mode Toggle: Seamlessly switch between dark and light themes
- Project Showcase: A curated selection of my projects, with descriptions and links
- Responsive Design: Optimized for various screen sizes and devices

Tech Stack

- Frontend: React
- Styling: Tailwind CSS
- Deployment: Netlify

Getting Started

1. Clone the repository: `git clone https://github.com/SabeeJeedorh/portfolio.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
